# Maven

To add a dependency on _Result_ using [**Maven**](https://maven.apache.org/), use the following:

```markup
<dependencies>
    <dependency>
        <groupId>com.leakyabstractions</groupId>
        <artifactId>result</artifactId>
        <version>{{ site.current_version }}</version>
        <type>pom</type>
    </dependency>
</dependencies>
```
